# LIA + API Meta Oficial (Atendimento de Recrutamento)

Responsável: Vinicius Barbarulo  

O QUE É:

Sustentação e evolução da LIA — agente de IA que realiza o primeiro atendimento automático de candidatos via WhatsApp Business (API Meta Oficial), integrado ao Chatwoot como CRM de atendimento e ao Uazapi para gestão das instâncias e chips.

  

STATUS ATUAL:

LIA em produção desde abril/2026. API Meta Oficial conectada. Automação de disparo de recrutamento revisada para API oficial. Sistema de controle de chips bloqueados operacional (tabela NocoDB). Monitoramento de instâncias desconectadas ativo.

  

RESULTADO ESPERADO:

Uptime ≥99% + SLA de primeira resposta <2min em horário comercial + redução mensurável no tempo de qualificação manual do time de Recrutamento.
